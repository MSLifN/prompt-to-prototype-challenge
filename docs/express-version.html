Express Version

Quick Version of the Challenge

Short on time? This express version uses custom GitHub Copilot agents and slash command prompts to guide you through a streamlined workflow from ideation to implementation in about 30-45 minutes.

The workflow is designed to keep your MVP achievable and focused on building with GitHub Copilot's free tier features.

What You'll Need

  • VS Code with GitHub Copilot extension installed
  • GitHub Copilot (free tier works perfectly)
  • Node.js 18+ installed on your system
  • The starter repository cloned locally or opened in GitHub Codespaces

Complete Workflow (6 Steps)

The easiest way to get started is using slash commands. Type / in GitHub Copilot Chat to see available commands:

Step 0: Setup (First Time Only)

If you're running locally, install dependencies first by typing this in Copilot Chat:

Command 
/install

Step 1: Ideate

Brainstorm and discover compelling MVP ideas through guided conversation:

Command 
/ideate

What happens:

  • Launches the @idea-generator agent
  • Asks about your interests, skill level, and goals
  • Generates 3-5 concrete, feasible ideas across different categories
  • Helps you refine and choose the best idea
  • Hands off to planning with a clear one-sentence goal

Step 2: Create PRD

Create a detailed implementation plan with a Product Requirements Document:

Command 
/prd

What happens:

  • Launches the @mvp-kickoff agent
  • Confirms the MVP goal and scope (~1 hour feasibility)
  • Identifies the ONE core user action
  • Breaks down UI components, state, and data requirements
  • Proposes 3-5 clear implementation steps
  • Generates a PRD.md file after your confirmation

Step 3: Build MVP

Build the MVP following the PRD steps:

Command 
/build

What happens:

  • Launches the @agent with pre-configured editing tools
  • Reads PRD.md to understand requirements
  • Implements the code step-by-step in src/App.jsx
  • Runs npm run dev to test
  • Verifies everything works with no console errors

Step 4: Start Server

Launch the development server to test your MVP:

Command 
/start

What happens:

  • Runs npm run dev in the terminal
  • Waits for the server to start
  • Provides the local URL (typically http://localhost:5173)
  • Confirms the server is ready

Step 5: Export Your Work

Download your MVP workspace as a ZIP file or push to your GitHub repository:

Command 
/export

What happens:

  • Guides you through opening the Command Palette
  • Using "Codespaces: Download Current Workspace" (if in Codespaces)
  • Saving the ZIP file with all your work
  • Explains what's included and next steps for deployment

Alternative: Using Agents Directly

You can also invoke agents directly in Copilot Chat if you prefer more control:

Ideation

Prompt 
@idea-generator I want to build something fun for my portfolio

Planning

Prompt 
@mvp-kickoff Build a Speed Reader app where users paste text and read words one-at-a-time

Implementation

Prompt 
@agent Implement the MVP plan from PRD.md

Or use inline chat (Ctrl+I or Cmd+I) directly in src/App.jsx to build step-by-step.

Tips for Success

  • Start with /ideate if you need help choosing what to build
  • Use slash commands for the simplest experience with pre-configured tools
  • Follow the sequence: /ideate ‚Üí /prd ‚Üí /build
  • Be specific: The more context you provide, the better the results
  • Iterate: Don't hesitate to ask follow-up questions or refine ideas
  • Use handoffs: Agents provide handoff buttons to seamlessly transition between phases
  • Keep scope minimal: The workflow is designed to keep your MVP achievable in ~1 hour

Express vs. Full Version

Aspect Express Version Full Version
Time 30-45 minutes 60+ minutes
Approach Automated workflow using slash commands and custom agents Manual step-by-step learning journey through documentation
Tools GitHub Copilot in VS Code with custom agents Copilot Chat in browser + GitHub Copilot in VS Code
Ideal For Quick prototyping, experienced developers, time-constrained sessions Learning AI-assisted development, understanding the full process
Interaction Type commands, follow agent guidance Copy/paste prompts, iterate with AI
