Express Version

Paid Tier Only

⚠️ Requires Paid GitHub Copilot

This Express Version uses custom agents and slash commands which require a paid GitHub Copilot subscription (Pro, Business, or Enterprise). For free tier users, please use the standard Step 6 guide instead.

Quick Version of the Challenge

The Express Version provides a streamlined, agent-powered workflow that automates much of the prototyping process using custom Copilot agents and slash commands.

Features (Paid Tier Only):

  • /ideate - Launch the idea generator agent
  • /prd - Create an implementation plan automatically
  • /build - Let the agent implement your code
  • /start - Start the development server
  • /export - Download or push your work

For detailed instructions on using the Express Version, see the README Express Version section.
