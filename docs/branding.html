Objective/Outcomes:

By completing this module, you will be able to:

  • Use your vibe words and research insights to shape visual and verbal branding
  • Generate visuals or name and tagline options with Copilot
  • Create a style palette that matches your product's personality
  • Write a clear description to guide consistent design choices
  • Generate a transparent background logo using Copilot
  • Understand how branding assets influence user experience

3.1 Get Started

Branding is more than a logo ‚Äî it's how people feel when they interact with your product.

In this step you'll:

  1. Decide what your product should communicate instantly
  2. Create visuals and words that support that feeling
  3. Package those into a brand kit you can apply to your prototype

3.2 Develop the Brand Identity

We'll utilize Copilot to help us craft a very high-level and lightweight Product Requirements Document.

Steps:
  1. Modify the below prompt as needed. As you are returning to your original conversation with Copilot that already has the context, but if you started here with no previous conversation, you will need to provide context around the project idea.
  2. Copy the prompt
  3. Continue your conversation and submit the prompt
Prompt 
Create a brand identity for my project.

Please provide:
1. 3 possible tagline options.
2. A suggested color palette (with hex codes) that matches my vibe words.
3. A short style description for use in design tools (e.g., "Bold geometric shapes with bright gradients").
4. A text prompt I can use to generate a simple, scalable logo with a transparent background.

Keep everything simple enough to create and apply within a 1-hour workshop build.

3.3 Create a Logo

We will use the text prompt the response created to create a logo image.

Steps:
  1. Copy the text prompt to create a logo generated from the previous response
  2. Modify if needed to suggest it should 'create a logo'. Submit the prompt
  3. Download the logo

3.4 Save Your Brand Identity

Save your brand identity details so you can reference them when building your prototype.

Steps
  1. Once Copilot has provided your brand identity (taglines, color palette, style description, and logo prompt), select all the text in the response (click and drag, or use Ctrl+A on Windows / Cmd+A on Mac)
  2. Copy the text (Ctrl+C on Windows / Cmd+C on Mac, or right-click > Copy)
  3. Open your preferred text editor or word processor where you saved your previous work (Word, Google Docs, Notepad/TextEdit, OneNote, Notion, etc.)
  4. Paste the brand identity (Ctrl+V on Windows / Cmd+V on Mac) into your document
  5. If you created a logo, download the logo image and save it in the same folder as your notes
  6. Keep this brand information handy‚Äîyou'll use the colors, tagline, and style description when building your prototype

Tip: Consider organizing your workshop outputs into sections within one document (Idea ‚Üí Research ‚Üí Branding ‚Üí Requirements ‚Üí Prototype Notes) for easy reference throughout the challenge.
