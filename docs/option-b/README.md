## üöÄ Getting Started

### Step 1: Choose a Template

Browse the templates above and pick one that interests you. Open it in VS Code.

### Step 2: Preview in Browser

Right-click the file and select "Open with Live Server" or simply open the HTML file in your browser.

### Step 3: Customize with Copilot

Open Copilot Chat (`Ctrl+Alt+I`) and ask for changes:

```
I'm working on this accessibility template. Help me:
- Add a new color scheme option
- Save user preferences to localStorage
- Add more keyboard shortcuts
```

### Step 4: Expand the Feature

Use inline edit (`Ctrl+I`) to modify specific sections:

```
Add a button to export these settings
```

Or ask for entirely new features:

```
Add a timer feature that reminds users to take breaks
```

## üé® Template Features

All templates include:

- **Skip to main content** link for screen reader users
- **Semantic HTML** with proper heading structure
- **ARIA labels** where appropriate
- **Keyboard navigation** support
- **Focus indicators** that are clearly visible
- **Responsive design** that works on mobile
- **localStorage** to save user preferences

## üîß Technical Notes

- Each template is a single HTML file (no build step required)
- CSS and JavaScript are embedded for simplicity
- Uses modern browser APIs (Web Speech API, localStorage)
- Works in all major browsers (Chrome, Firefox, Edge, Safari)
- No external dependencies
