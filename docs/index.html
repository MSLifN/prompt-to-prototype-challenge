About the Challenge

Prompt to Prototype is a fast-paced Artificial Intelligence (AI)-powered creation challenge. In one hour, you'll take an idea from a concept to a working web-based prototype using Copilot and GitHub Copilot.

You'll go through six key stages:

Learning Objectives / Outcomes

By completing this exercise, you will strengthen your ability to:

  • Generate and refine ideas using structured prompts and creative constraints
  • Conduct targeted research to explore a domain, validate an idea, and identify potential gaps
  • Perform a high-level market and competitive analysis using AI tools to synthesize insights quickly
  • Translate ideas into clear product requirements that guide a build
  • Develop a brand identity that communicates your product's vibe and value
  • Use AI as a creative and technical collaborator â€” from research and ideation to code generation and design
  • Scope an MVP effectively so you can deliver something functional within a short timeframe
  • Prototype rapidly with GitHub Copilot-assisted development
  • Package and present your work in a way that's compelling to share

What You'll Need

Before you begin, make sure you have:

  • Copilot Chat in the browser at https://m365.cloud.microsoft/chat/
  • AI Powered Prototyping tool such as GitHub Copilot in Visual Studio Code.
  • Screen Capture Software such as Snippet

Expectations & Output

By the end of this session you should have:

  • A scoped, realistic MVP idea
  • Research summary (market insights, competitors, audience)
  • A basic PRD (must-have + nice-to-have features)
  • Brand identity (name, tagline, color palette, logo)
  • A working prototype (coded MVP)

Tips for Success

  • Scope small first â€” the goal is to finish something within the workshop time
