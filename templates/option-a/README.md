## üöÄ Getting Started

### Step 1: Open in VS Code

1. Open VS Code
2. Go to **File > Open Folder** and select the `templates/option-a` folder
3. Open `index.html`

### Step 2: Open Copilot Chat

Click the chat icon in the sidebar, or press `Ctrl+Alt+I` (Windows) / `Cmd+Alt+I` (Mac)

### Step 3: Integrate Your PRD

Copy your PRD into the chat and ask:

```
Based on this PRD, update the index.html to create the main UI for my prototype:

[Paste your PRD here]
```

### Step 4: Apply Your Branding

```
Apply these brand colors and styling to the page:
- Primary color: [your color]
- Secondary color: [your color]
- Font: [your font choice]
- Update the header with my product name: [name]
- Update the tagline to: [tagline]
```

### Step 5: Build Features

For each feature in your PRD, ask Copilot:

```
Add [feature name] to the page. It should [describe what it does].
```

### Step 6: Preview Your Work

Simply open `index.html` in your web browser:
- **Windows:** Right-click the file > Open with > Chrome/Edge/Firefox
- **Mac:** Right-click > Open With > Safari/Chrome
- **Or:** Drag the file into your browser window

## ‚ôø Accessibility Tips

The template includes accessibility basics:
- Skip link for keyboard users
- Semantic HTML (`<header>`, `<main>`, `<section>`)
- ARIA labels on sections
- Focus indicators on buttons

Ask Copilot to help with more:
```
Review this HTML for accessibility issues and suggest improvements
```
