## ðŸš€ Getting Started

### Step 1: Choose a Template

Browse the templates above and pick one that fits your goal. Open it in VS Code.

### Step 2: Open Copilot Chat

Click the chat icon in the sidebar, or press `Ctrl+Alt+I`

### Step 3: Introduce Your Project

```
I'm creating a [type of document] for my project. Here's my idea:

[Paste your PRD or describe your idea]

Help me fill in this template with compelling content.
```

### Step 4: Work Section by Section

For each section, ask Copilot to help:

```
Write the [section name] section. It should [describe what you want].
```

### Step 5: Refine with Inline Edit

Select any text and press `Ctrl+I` to ask for improvements:

```
Make this clearer and more concise
```

```
Add more detail about [specific aspect]
```

```
Adjust the tone to be more [professional/friendly/persuasive]
```
